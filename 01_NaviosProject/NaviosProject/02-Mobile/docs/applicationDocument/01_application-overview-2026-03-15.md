# Application Overview

**Project**: NaviOs
**Date**: 2026-03-15
**Version**: MVP (Pre-release)

---

## 1. Purpose

NaviOs は**地域情報をリアルタイムに共有する**モバイルアプリ。
近くのユーザー同士が「物資情報」「イベント」「助け合い」「行政情報」を投稿・閲覧し、
位置情報ベースのつぶやき（Talk）で気軽にコミュニケーションできる。

**コアバリュー**: 今いる場所の "生きた情報" に、誰でもアクセスできる

---

| Segment | Description |

## 4. User Flow

```
アプリ起動
  │
  ├─ [ゲスト] ─→ タブ画面（Talk / Navios / Map / Feed / My）
  │                │
  │                ├─ 閲覧のみ可能（投稿詳細、コメント閲覧、検索）
  │                ├─ 投稿作成 → ログイン画面へリダイレクト
  │                ├─ Talk投稿 → 「ログインして参加する」バー表示
  │                └─ My → ログイン/登録誘導画面
  │
  └─ [ログイン済み] ─→ タブ画面（全機能利用可能）
                       │
                       ├─ Talk: つぶやき投稿 / リアクション / 返信
                       ├─ Navios: AI検索
                       ├─ Map: 近くの投稿閲覧
                       ├─ Feed: タイムライン閲覧
                       ├─ My: プロフィール編集 / 投稿管理 / FABで新規投稿
                       └─ 投稿作成: 3ステップウィザード → 成功画面
```

## 5. Non-functional Constraints

| Constraint | Detail |
|------------|--------|
| オフライン | 未対応。ネットワーク接続必須 |
| パフォーマンス | 投稿一覧は50件制限。RPC距離検索は500m半径 |
| セキュリティ | Supabase RLS有効。API keyはクライアント公開キーのみ |
| 画像サイズ | 投稿画像800px / アバター400px / quality 0.7 に自動圧縮 |
| Talk有効期限 | つぶやきは24時間で自動消滅（DB側 expires_at） |
| 位置情報 | foreground permission のみ。バックグラウンド追跡なし |
| 対応言語 | 日本語のみ |
| 対応OS | iOS / Android（Expo managed workflow） |
