# 05_改修履歴

最終更新: 2026-02-16

## このフォルダについて

Naviosプロジェクトの機能追加やUI改善など、各フェーズでの改修内容の詳細記録が格納されています。

## ファイル一覧

### Phase1_Vol1_UI_UX改修.md

Phase1完了後のUI/UX品質向上改修の詳細記録です。

**主な改修内容:**

1. **検索機能の修正**
   - 全投稿から検索可能に変更（従来はフィルター結果内のみ）

2. **モバイルフィルターUI移動**
   - フィルターをナビゲーションドロワーから検索ボックス下に移動
   - ガラスモーフィズムデザインの採用

3. **UI/UXデザインポリッシュ**
   - Tailwindカスタムカラー定義
   - ガラスモーフィズム効果の追加
   - ピル型UIへの統一
   - フォント改善

4. **追加改修**
   - モバイルフィルターバーのスクロールバー非表示
   - 現在地マーカーのアニメーション追加
   - 全体表示を現在位置基準に変更
   - イベントピンのカテゴリ優先デザイン

5. **地図SDK差し替え可能設計**
   - LeafletとGoogle Maps SDKを切り替え可能な設計に変更

6. **投稿完了画面の追加**
   - 投稿後の導線改善

7. **イベントアーカイブ機能**
   - 終了後24時間を超えたイベントを自動アーカイブ
   - 過去人気イベント表示機能

8. **閲覧数機能（view_count）**
   - イベント詳細ページの閲覧数を記録・表示

9. **住所自動補完（逆ジオコーディング）**
   - 緯度経度から住所を自動取得

**変更ファイル数**: 20ファイル以上

**検証結果**: ビルド成功、主要導線に問題なし
