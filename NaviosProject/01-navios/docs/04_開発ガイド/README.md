# 04_開発ガイド

最終更新: 2026-02-16

## このフォルダについて

Naviosの開発を進めるための実践的なガイドが格納されています。

## ファイル一覧

### 環境構築ガイド.md

開発環境をゼロから構築するための手順書です。

**内容:**
- 前提条件（必須ソフトウェア）
- セットアップ手順
- 環境変数の設定
- トラブルシューティング

**こんな人におすすめ:**
- 初めてプロジェクトに参加する人
- 開発環境を再構築したい人

### コマンド一覧.md

開発で使用する全コマンドの参考書です。

**内容:**
- 基本コマンド
- ビルド・テスト
- データベース操作
- クリーンアップ
- Git操作
- トラブルシューティング

**こんな人におすすめ:**
- よく使うコマンドをすぐに確認したい
- データベース操作方法を知りたい
- 問題が起きたときの対処法を探している

### スクレイピング導入ガイド.md

KCICイベントをCSV化してSupabaseへ投入する運用手順です。

**内容:**
- スクレイピング実行方法
- CSV生成と投入コマンド
- よくあるエラーと対処

**こんな人におすすめ:**
- 外部イベント情報を一括取り込みしたい人
- Supabase投入を再現可能な手順で運用したい人

## クイックスタート

### 初めて開発環境を作る

1. [環境構築ガイド.md] を最初から順番に読む
2. 手順に従ってセットアップ
3. `npm run dev` で開発サーバーを起動
4. ブラウザで `http://localhost:3000` を開く

### コマンドをすぐに調べたい

[コマンド一覧.md] の目次から必要な項目を探してください。

### スクレイピング運用をしたい

[スクレイピング導入ガイド.md] を参照して、  
`KCIC -> CSV -> Supabase` の順で実行してください。

## よく使うコマンド（抜粋）

```bash
# 開発サーバー起動
npm run dev

# コードチェック
npm run lint

# ビルド
npm run build

# データベースリセット
npm run prisma:reset
```

## トラブルシューティング

問題が発生した場合:

1. [コマンド一覧.md] の「トラブルシューティング」セクションを確認
2. [環境構築ガイド.md] の「トラブルシューティング」セクションを確認
3. それでも解決しない場合はチームに相談
