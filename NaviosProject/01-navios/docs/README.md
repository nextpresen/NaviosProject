# Navios プロジェクトドキュメント

最終更新: 2026-02-16

## このドキュメントについて

Naviosは地域の「今使える情報」を地図中心で発見・投稿できる生活支援Webアプリケーションです。

このドキュメント集は、開発者だけでなく、パソコン初心者の方でも理解できるよう、番号順に読み進められる構成になっています。

## ドキュメントの構成

フォルダ名の番号順に読むことで、プロジェクト全体を理解できます。

### 📖 01_はじめに

プロジェクトの概要と、開発を始めるための基本情報が入っています。

- プロジェクト概要
- 技術スタックの紹介
- 環境セットアップ手順

**初めての方は、ここから読んでください。**

### 📐 02_設計書

システムの設計に関するドキュメントです。全体像やフェーズごとの開発内容が記載されています。

- 全体設計（アーキテクチャ、技術選定）
- Phase1設計書（基本機能の設計）

### 💾 03_データベース

データベースの設計書です。どのようなデータを、どのように保存しているかが分かります。

- データベース設計書
- テーブル構造の説明

### 🛠️ 04_開発ガイド

実際に開発を進めるための手順やコマンドが記載されています。

- 開発環境の構築方法
- よく使うコマンド一覧
- テスト実行方法

### 📝 05_改修履歴

機能追加やUI改善など、各フェーズでの改修内容の詳細記録です。

- UI/UX改修（Phase1 Vol.1）
- 機能追加の記録

### 🔧 06_リファクタリング

コードの品質向上やメンテナンス性改善のための計画と実施記録です。

- リファクタリング計画
- 実施チェックリスト

### 🤖 99_AI向け

AIエージェント（Claude Code等）が効率的にプロジェクトを理解するための要約ドキュメントです。

- AI_DESIGN.md（システム全体の要約）
