# モバイル

最終更新: 2026-02-16

## このフォルダについて

Naviosのモバイル開発に関する方針と、開発開始時に必要な引き継ぎ資料を格納しています。

## ファイル一覧

### モバイル開発.md

モバイル開発の全体方針です。

- Web: Leaflet（閲覧中心）
- Mobile: Google Maps API（投稿体験強化）
- コスト設計と最短実装ロードマップ

### 引き継ぎ書.md

モバイル実装に入る前の必読資料です。

- `place_name` / `address_label` 契約先行改修の内容
- API契約（POST/PUT）の注意点
- 着手時チェックリスト

## 読む順番

1. `モバイル開発.md` で方針を把握
2. `引き継ぎ書.md` で実装前提を確認
3. モバイル実装へ着手
